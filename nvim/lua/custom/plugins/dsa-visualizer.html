🔧 Neovim Integration Setup

Add this to your Neovim configuration to integrate the visualizer:

-- ~/.config/nvim/lua/dsa-visualizer.lua local M = {} function M.visualize_current_problem() local current_line = vim.api.nvim_get_current_line() local problem = vim.fn.input("DSA Problem to visualize: ", current_line) if problem ~= "" then local url = "http://localhost:3000?query=" .. vim.fn.escape(problem, " &") vim.fn.system("open '" .. url .. "'") -- macOS -- vim.fn.system("xdg-open '" .. url .. "'") -- Linux -- vim.fn.system("start '" .. url .. "'") -- Windows end end -- Key mapping vim.keymap.set('n', 'dv', M.visualize_current_problem, { desc = 'Visualize DSA problem' }) return M

Then add this to your init.lua:

require('dsa-visualizer')

Usage: Press <leader>dv in Neovim to visualize the current line/problem!
